1. 인공지능(AI, Artificial Intelligence)
컴퓨터나 기계가 인간처럼 생각하고 학습하며 문제를 해결하는 능력을 갖추도록 만드는 기술
컴퓨터 과학의 한 분야로, 데이터를 활용해서 의사결정을 내리거나 예측하는 시스템을 개발하는 것을 목표로 한다.
AI; Artificial Intelligence > ML; Machine Learning > Deep Learning

1-1. 규칙 기반 알고리즘(Rule-based algorithms)
명시적으로 정의된 규칙(If-Then-Else 문법 등)을 사용하여 특정 문제를 해결하거나 작업을 수행하는 알고리즘
이러한 알고리즘은 사람이 직접 작성한 논리와 조건에 따라 작동하며, 데이터를 학습하지 않고도 결과를 도출할 수 있다.
개발자의 생각이 담겨있어, 프로그램적으로 문제를 해결하는 알고리즘
예: 과거의 로봇청소기 - 센서를 달아서 어떤 신호가 오게되면 어떻게 움직이고 헤결할지 개발자가 판단

1-2. 머신러닝(Machine Learning)
인공지능(AI)의 한 분야로, 컴퓨터가 명시적으로 프로그래밍되지 않아도 데이터에서 패턴을 학습하고, 이를 기반으로 새로운 데이터를 예측하거나 결정을 내리는 기술
데이터에 의해서 학습이 된다.
머신러닝은 AI의 핵심 기술로, 우리가 사용하는 많은 기술과 서비스의 기초가 된다.

1-3. 딥러닝(Deep Learning)
머신러닝의 한 분야로, 사람의 뇌(뉴런)에서 영감을 받은 인공신경망(Artificial Neural Networks)을 사용하여 데이터를 학습하고 복잡한 문제를 해결하는 기술
특히, 딥러닝은 대량의 데이터와 고성능 컴퓨팅 자원을 활용하여 뛰어난 성능을 발휘하며, 이미지, 음성, 자연어 처리 등에서 주로 사용된다.

2. 머신러닝을 먼저 공부해야 하는 이유
2-1. 기초 개념 없이 딥러닝은 어렵다
딥러닝은 머신러닝의 하위 분야로, 기본적인 개념(예: 과적합, 분산과 편향, 학습/검증/테스트 분리, 평가 지표 등)을 머신러닝에서 먼저 익히지 않으면 딥러닝 모델의 작동 원리나 결과를 해석하기 어렵다.

2-2. 작고 단순한 문제에는 머신러닝이 더 효과적
모든 문제에 딥러닝이 필요한 것은 아니다. - 비싸고 큰 하드웨어 + 많은 운영 비용 + 많은 인력 필요
단순한 문제에서는 로지스틱 회귀나 결정 트리 같은 머신러닝 모델이 빠르고 정확하며, 데이터도 적게 필요하다.
예: 고객 이탈 예측이나 단순한 이진 분류 문제

3. 머신러닝으로 할 수 있는 일
분류 (Classification)
    스팸 메일 분류, 암 진단 (양성/악성), 고객 이탈 예측, 고장 여부 판단
    스팸 메일 분류 - 과거에는 사람이 코딩으로 분류했다면, 지금은 스팸함 데이터를 모아서 기계가 학습/분류
회귀 (Regression); 예측;
    주택 가격 예측, 주가 예측, 광고 클릭률 예측, 날씨 온도 예측
군집화 (Clustering); 비지도 학습(레이블/답이 없다) -> 패턴 발견 후 군집화
    고객 세그먼트 분류, 상품 자동 그룹화, 이상 거래 탐지(불량률, 금융), 문서 주제 군집화
추천 시스템; 맞춤형 추천
    영화, 음악, 상품 추천, 사용자 맞춤 콘텐츠 제공
자연어 처리 (NLP); 기계가 사람의 언어를 읽고 이해하는 것(임베딩)
    문장 감정 분석, 뉴스 기사 분류, 챗봇 응답 생성, 기계 번역
이미지 처리 (Computer Vision)
    얼굴 인식, 객체 탐지, 자율주행 자동차 인식, 의료(세포), X-ray 진단 보조
이상 탐지 (Anomaly Detection)
    금융 사기 탐지, 네트워크 이상 징후 감지, 공정 불량 탐지
강화 학습 (Reinforcement Learning); 점수(score)
    게임 AI (알파고 등), 로봇 제어, 스마트 팩토리 자동 제어
시계열 분석 - 회귀 분석
    수요 예측, 재고 관리, 에너지 사용량 예측
음성 인식 및 합성 - 자연어처리
    음성 명령 인식, AI 스피커, 텍스트 음성 변환 (TTS)
